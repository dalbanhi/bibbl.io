## About The Project

Bibbl.io is a book/reading organizer. Its goal is to help students, academics, and bookworms organize their reading priorities and keep a more detailed log of their reading by allowing users to organize books by reading status and “shelves” and keep better track of their reading habits.

## Setting up and Running the Project

To get a local copy up and running follow these simple example steps.

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo (or download zip file)
   ```sh
   git clone https://github.com/dalbanhi/bibbl.io.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Install requirements.txt
4. Run 
   ```sh 
    python manage.py makemigrations
   ```
5. Run 
   ```sh 
    python manage.py migrate
   ```
6. Run 
   ```sh 
    python manage.py runserver
   ```
